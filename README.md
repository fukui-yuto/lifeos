# LifeOS

Windows 向けの生活管理デスクトップアプリです。付箋ノート・タスク管理・目標管理・操作ログをひとつのアプリで管理できます。

---

## 機能

### 付箋ノード
- デスクトップ上に自由配置できる付箋ウィンドウ
- タイトル・メモ・チェックリストを1枚の付箋で管理
- 20色のカラーパレットから色を変更（右クリック）
- メモ・チェックリストのセクションを折りたたみ可能
- GridSplitter でメモとチェックリストの高さを自由に調整
- ウィンドウのサイズ・位置は自動保存

### タスク管理
- 全付箋のチェックリストを一覧表示
- チェック・編集・削除が可能
- 完了タスクは打ち消し線で表示

### 目標管理
- 目標のタイトル・説明・期限を管理
- 付箋ノートを目標に紐づけ（多対多）
- 完了フラグで達成状況を管理

### 操作ログ
- ノート・タスク・目標の追加・更新・削除を自動記録
- タイムスタンプ・操作種別・詳細を一覧表示

### システムトレイ
- 最小化してもトレイに常駐
- トレイアイコンクリックで管理ウィンドウを前面に表示
- Windows 起動時に自動起動

## インストール

### 方法 1 — インストーラーを使う（推奨）

1. `LifeOS.exe` と `Install.ps1` を同じフォルダに置く
2. `Install.ps1` を右クリック →「PowerShell で実行」
3. デスクトップとスタートメニューにショートカットが作成される
4. アプリ起動後、Windows 起動時に自動起動するよう登録される

> 管理者権限は不要です。データは `%LOCALAPPDATA%\LifeOS\` に保存されます。

### 方法 2 — exe を直接実行

`LifeOS.exe` をダブルクリックするだけで起動します。インストール不要。

## 操作方法

### 付箋の基本操作

| 操作 | 方法 |
|------|------|
| 付箋を追加 | 管理ウィンドウ「ノートボード」→「＋ ノート追加」 |
| 付箋を開く | ノートボードのカードをダブルクリック |
| 付箋を移動 | タイトルバーをドラッグ |
| 付箋をリサイズ | 右下角をドラッグ |
| 色を変更 | 付箋上で右クリック → 色を選択 |
| 付箋を閉じる | タイトルバーの「✕」ボタン |
| 付箋を削除 | ノートボードのカード右上「🗑」ボタン |
| 管理ウィンドウを開く | 付箋タイトルバーの「⊞」ボタン、またはダブルクリック |

### チェックリスト

| 操作 | 方法 |
|------|------|
| タスク追加 | 付箋下部の入力欄に入力して Enter |
| タスク完了 | チェックボックスをクリック |
| タスク編集 | タスクテキストをクリックして編集、Enter または フォーカスアウトで保存 |
| タスク削除 | タスク右端の「✕」ボタン |
| セクション折りたたみ | 「▼」ボタンで折りたたみ、「▶」で展開 |

### 目標管理

| 操作 | 方法 |
|------|------|
| 目標追加 | 「目標」タブ → 「＋ 目標追加」 |
| 目標編集 | 目標カードの「編集」ボタン |
| 付箋と紐づけ | 編集ダイアログのリストから付箋を選択（複数選択可） |
| 目標削除 | 目標カードの「削除」ボタン |

## ビルド方法

### 前提条件

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8)

### 開発用ビルド・実行

```bash
dotnet run --configuration Release
```

### 配布用 exe 作成

```bash
dotnet publish --configuration Release --runtime win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true --output ./publish
```

`publish/LifeOS.exe` が生成されます（単一ファイル・.NET ランタイム同梱）。
